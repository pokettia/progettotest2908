Methodology by Example: ArCo & Cultural Heritage

1. Project Idea

Use a knowledge graph (any) as subject of the project. Here, the ArCo Knowledge Graph is used as an example to showcase methodology: discover a topic, run SPARQL queries, use LLMs for enrichment, verify results, and publish findings on a GitHub-based website.

2. ArCo Ontology Overview

  • ArCo is a network of ontologies for Italian cultural heritage.
  • Core ontology: general concepts (Agent, CulturalEntity, Location).
  • Denotative Description: measurable/cataloguing attributes (dimension, material, technique).
  • Context Description: contextual attributes (authorship, subject, conservation, iconclass).
  • Other modules cover specific domains (events, instruments, etc.).

3. Workflow / Methodology

  1. Explore ontologies + KG (ArCo docs, modules).
  2. Identify useful classes/properties (e.g. a-cd:Subject, a-cd:hasSubject).
  3. Form SPARQL queries against ArCo endpoint.
  4. Select a topic (e.g. artworks with same subject or iconclass).
  5. Use LLMs to extract information & propose triples.
  6. Verify if information exists in ArCo before adding.
  7. Create enriched triples and document provenance.
  8. Publish results on GitHub Pages.

4. Example Query: Tondo Doni (Michelangelo)

PREFIX arco: <https://w3id.org/arco/ontology/arco/>
SELECT DISTINCT ?x ?l ?p ?o
WHERE {
  ?x a arco:HistoricOrArtisticProperty ;
     rdfs:label ?l ;
     ?p ?o .
  FILTER(regex(str(?l), "tondo doni", "i"))
}

5. Subjects in ArCo (Q1)

PREFIX a-cd: <https://w3id.org/arco/ontology/contextdescription/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT ?subject ?label
WHERE {
  ?subject a a-cd:Subject ;
           rdfs:label ?label .
}
LIMIT 20

6. Artworks Sharing Same Iconclass (Q2)

PREFIX a-cd: <https://w3id.org/arco/ontology/contextdescription/>
PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT ?cp1 ?cp1label ?cp2 ?cp2label ?code
WHERE {
  ?cp1 a arco:CulturalProperty ;
       a-cd:iconclassCode ?code ;
       rdfs:label ?cp1label .
  ?cp2 a arco:CulturalProperty ;
       a-cd:iconclassCode ?code ;
       rdfs:label ?cp2label .
  FILTER(?cp1 != ?cp2)
  FILTER(str(?code) != "")
}
LIMIT 100

7. RDF Example (using ArCo Ontology)

@prefix arco: <https://w3id.org/arco/ontology/arco/> .
@prefix a-cd: <https://w3id.org/arco/ontology/contextdescription/> .
@prefix ex: <http://example.org/> .
@prefix rdfs: <http://www.w3.org/2000/01/rdf-schema#> .

ex:Tondo_Doni a arco:CulturalProperty ;
  a-cd:hasAuthor "Michelangelo" ;
  a-cd:hasSubject ex:Sacra_Famiglia ;
  rdfs:label "Tondo Doni" .

ex:Sacra_Famiglia a a-cd:Subject ;
  rdfs:label "Sacra Famiglia" .
